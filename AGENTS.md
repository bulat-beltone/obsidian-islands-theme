Use mcp "obsidian-live" to interact with Obsidian HTML and CSS.

When making git commits, do NOT add "Co-authored-by: Cursor" or any Cursor co-author trailer to commit messages.

Commit messages описывают только изменения в теме (theme.css). Изменения в других файлах (AGENTS.md, .gitignore, README и т.д.) не упоминаются, если в том же коммите есть изменения темы. Исключение — когда theme.css не менялся.

`My Notes/` — личные заметки о разработке темы (dev setup, исследования, TODO). Папка в `.gitignore` и не попадает в публичный репо. Сохраняй сюда всё, что нужно запомнить, но не нужно публиковать вместе с темой.
